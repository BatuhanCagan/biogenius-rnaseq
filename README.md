<h2>📌 Overview</h2>
<p>BioGenius is a production-ready, interactive web application designed to bridge the gap between complex bioinformatics algorithms and user-friendly data analysis. It allows researchers to perform <strong>Differential Gene Expression (DGE)</strong> analysis on RNA-seq data directly from their browser, eliminating the need for complex R pipelines.</p>
<p>Under the hood, it leverages the mathematical rigor of <strong>PyDESeq2</strong> (Negative Binomial GLMs) while providing a highly optimized, state-managed frontend built with <strong>Streamlit</strong> and <strong>Plotly</strong>.</p>

<h2>✨ Key Engineering Features</h2>
<ul>
  <li><strong>Statistical Rigor:</strong> Implements the industry-standard DESeq2 algorithm natively in Python.</li>
  <li><strong>Optimized Performance:</strong> Utilizes caching (<code>@st.cache_data</code>) and session state management to prevent redundant heavy matrix computations. Matrix orientation logic is handled automatically.</li>
  <li><strong>Interactive Visualization:</strong> Dynamic, hoverable Volcano Plots replacing static, hard-to-read academic charts.</li>
  <li><strong>Global-Ready Architecture (i18n):</strong> Clean separation of concerns with a dictionary-based localization system (English/Turkish) using external JSON files.</li>
  <li><strong>Built-in Mock Data Engine:</strong> Includes a synthetic data generator that simulates true biological variance (Poisson noise) and injects actual biological "easter eggs" (e.g., upregulated <em>BRCA1</em> and downregulated <em>TP53</em>).</li>
</ul>
